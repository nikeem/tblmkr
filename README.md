# TblMaker

Конвертер текстовых данных в таблицы для Creatium.

## Описание

Скрипт парсит текстовый файл с табулированными данными и генерирует JSON-код для виджета таблицы Creatium.

## Использование

```bash
python3 tblmaker.py
```

## Формат входного файла (in.txt)

```
Номер	Фамилия, имя	Амплуа	Дата рождения	Гражданство
35	Кульбаков Иван	вратарь	18.09.1996	Беларусь
72	Костин Денис	вратарь	21.06.1995	Россия
...
Главный тренер: Исаков Алексей
```

- Первая строка — заголовки
- Данные разделены табами
- Последняя строка — тренер (опционально)

## Особенности

- Автоматическое сокращение амплуа:
  - вратарь → вр.
  - защитник → защ.
  - нападающий → нап.
- Тренер добавляется в конец таблицы с пустыми полями номера и даты рождения
- Стабильное форматирование HTML

## Структура проекта

```
tblmaker/
├── tblmaker.py          # Основной скрипт
├── template.json        # JSON шаблон Creatium
├── extract_template.py  # Утилита для создания шаблона
├── in.txt              # Входные данные
└── out.txt             # Сгенерированный JSON
```
